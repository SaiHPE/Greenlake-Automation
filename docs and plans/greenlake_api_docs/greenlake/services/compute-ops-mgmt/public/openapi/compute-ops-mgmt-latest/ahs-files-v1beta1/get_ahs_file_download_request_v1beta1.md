# Download parsed AHS file contents

This API triggers a get request to download parsed (CSV/JSON) AHS file contents.

Endpoint: GET /compute-ops-mgmt/v1beta1/ahs-files/{id}/download
Version: latest
Security: Bearer

## Path parameters:

  - `id` (string, required)
    Unique File Identifier (File UUID)

## Query parameters:

  - `filename` (string, required)
    File name to be downloaded
    Example: "MXQ0xxxT3-20241205-1025-SERVER_INVENTORY.json"
