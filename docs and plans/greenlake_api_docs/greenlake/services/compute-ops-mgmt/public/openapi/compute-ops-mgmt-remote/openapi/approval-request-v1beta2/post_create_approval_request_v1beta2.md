# Approve request

Approves or declines an approval request

Endpoint: POST /compute-ops-mgmt/v1beta2/approval-requests/{request_id}/approve
Version: latest
Security: Bearer

## Path parameters:

  - `request_id` (string, required)
    Unique Request Identifier

## Response 200 fields (application/json):

  - `approvalState` (string)
    The state set by the user when approving or declining an approval request.
    Enum: "APPROVED", "DECLINED"

  - `remarks` (string)
    Comments on the approval
