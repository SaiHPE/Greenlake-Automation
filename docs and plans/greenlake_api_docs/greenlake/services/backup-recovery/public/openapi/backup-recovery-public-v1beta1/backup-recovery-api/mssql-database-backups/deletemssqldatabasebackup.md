# Delete an MSSQL database backup.

Remove an MSSQL database backup.

Endpoint: DELETE /backup-recovery/v1beta1/mssql-databases/{db-id}/backups/{backup-id}
Version: 1.1.0
Security: bearer

## Path parameters:

  - `db-id` (string, required)
    UUID string uniquely identifying the MSSQL Database
    Example: "9b4c14a6-3cd5-4907-97c4-cf44c5b641e4"

  - `backup-id` (string, required)
    UUID string uniquely identifying the backup.
    Example: "9b4c14a6-3cd5-4907-97c4-cf44c5b641e4"
