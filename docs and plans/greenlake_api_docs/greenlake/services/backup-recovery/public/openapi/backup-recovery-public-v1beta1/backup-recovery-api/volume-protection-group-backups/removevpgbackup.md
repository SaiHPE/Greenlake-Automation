# Remove a Volume Protection Group backup.

Remove a Volume Protection Group backup.

Endpoint: DELETE /backup-recovery/v1beta1/volume-protection-groups/{id}/backups/{backup-id}
Version: 1.1.0
Security: bearer

## Path parameters:

  - `id` (string, required)
    UUID string uniquely identifying the Volume Protection Group.
    Example: "9b4c14a6-3cd5-4907-97c4-cf44c5b641e4"

  - `backup-id` (string, required)
    UUID string uniquely identifying the Volume Protection Group backup.
    Example: "9b4c14a6-3cd5-4907-97c4-cf44c5b641e4"
