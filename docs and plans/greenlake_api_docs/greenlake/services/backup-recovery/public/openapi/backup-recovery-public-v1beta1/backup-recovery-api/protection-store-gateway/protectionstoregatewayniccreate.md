# Creates a network interface on the Protection Store Gateway

Create network interface on the Protection Store Gateway

Endpoint: POST /backup-recovery/v1beta1/protection-store-gateways/{id}/createNic
Version: 1.1.0
Security: bearer

## Path parameters:

  - `id` (string, required)
    The UUID of the object
    Example: "c1a0eb78-41a0-4151-93b2-f057ffeca3f3"

## Request fields (application/json):

  - `nic` (object)

  - `nic.gateway` (string)
    Gateway associated with the network interface.

  - `nic.networkAddress` (string)
    IP address associated with the network interface.

  - `nic.networkName` (string)
    Name of the hypervisor network.

  - `nic.networkType` (string)
    Type of the network
    Enum: "DHCP", "STATIC"

  - `nic.subnetMask` (string)
    Subnet mask.
