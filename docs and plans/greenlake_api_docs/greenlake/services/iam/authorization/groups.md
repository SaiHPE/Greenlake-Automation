# Workspace Groups Management

The Workspace Groups Management service enables you to create, manage, and organize workspace groups within the HPE GreenLake platform. This service provides comprehensive CRUD (Create, Read, Update, Delete) operations for workspace groups management.

## Overview

This service allows you to:

- Create new workspace groups
- Retrieve existing groups in a workspace
- Update workspace group properties
- Delete workspace groups
- Add workspaces to groups
- Remove workspaces from groups
- Retrieve workspace details by groups
- Retrieve details for a workspace group


## API Version

Current API version: **v1beta1**

## Authentication

This API uses Bearer token authentication. Include your access token in the Authorization header:


```http
Authorization: Bearer <your-access-token>
```

## Base URL


```text
https://global.api.greenlake.hpe.com
```

## Getting Started

To get started with the Workspace Groups Management API:

1. Obtain an access token through the HPE GreenLake authentication system
2. Review the API reference documentation below
3. Start making API calls to manage your workspace groups
