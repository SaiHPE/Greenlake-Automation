# HPE GreenLake for Wellness glossary

## Acronyms

## UUID

Universally unique identifier (UUID)— This is a unique value used in computer systems to identify an object or item. Wellness events are automatically assigned a UUID, for example, `5d9eb55a28c7eb0001f472eb`.

## Terms

### Wellness event

Wellness events provide health insights about your HPE products and services and show information about automatically created support cases. Wellness events allow you to:

- Identify potential vulnerabilities before they affect your environment.
- Proactively monitor the health of your HPE products and services.
- View AI-powered recommendations about events.


### Case number

Every case is assigned a unique numeric identifier. Wellness events can lead to the creation of support cases for HPE support. Your organization can enable automatic case creation, meaning many wellness events will automatically create a support case. You can also manually create support cases to be reviewed by HPE support.

### Product

Logical grouping of products based on product line.

Example:

- `HPE Alletra 9000`
- `HPE Alletra 6000`
- `HPE Alletra 5000`


### Severity

Wellness events are categorized by the following statuses:

| Status | Description |
|  --- | --- |
| Critical | Urgent. This event needs immediate attention (for example, data loss or hardware damage might occur). |
| Warning | This event might require attention soon (for example, if a volume is running out of space). |
| Notice | This event needs to be reviewed but might not need any action. |
