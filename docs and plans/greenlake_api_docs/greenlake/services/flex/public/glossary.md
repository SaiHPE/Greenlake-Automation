# HPE Flex Solutions glossary

## Order

A Flex order represents a billing arrangement that includes information such as
billing account, SOW (Statement of Work), customer, reseller, distributor, and
partner details along with order start and end dates.

## Device

A Flex device represents a physical or managed device associated with a Flex order.
Devices include attributes such as MAC address, serial number, part number, device
type, model, and make.

## Billing account

An account used for billing purposes, identified by a unique billing account ID
and name.

## SOW (Statement of Work)

A unique identifier for a billing order, representing the statement of work
associated with a Flex order.

## Device type

The classification of a device. Supported types include: `ALS`, `AP`, `BLE`, `COMPUTE`, `CONTROLLER`, `DHCI_COMPUTE`, `DHCI_STORAGE`, `EINAR`, `EINR`, `GATEWAY`, `IAP`, `LTE_MODEM`, `MC`, `STORAGE`, `SWITCH`, `NW_THIRD_PARTY`, `PCE`, `SD_WAN_GW`, `OPSRAMP_SAAS`, `SD_SAAS`, `SENSOR`, `BRIDGES`, and `UNKNOWN`.

## Tag

A key-value pair of metadata applied to a device resource, primarily used to
categorize resources.
