# Delete replication partner from HPE Alletra Storage MP B10000 storage system identified by {systemId}

Delete replication partner from HPE Alletra Storage MP B10000 storage system identified by {systemId}

Endpoint: POST /storage-fleet/v1alpha1/devtype4-storage-systems/{systemId}/replication-partners/remove
Version: 1.2.0
Security: bearer

## Path parameters:

  - `systemId` (string, required)
    systemId of the storage system
    Example: "7CE751P312"

## Request fields (application/json):

  - `replicationPartners` (array, required)
    List of replication partner pairs to be deleted

  - `replicationPartners.srcReplicationId` (string, required)
    Id of source replication partner
    Example: "afb4961e47212e5bc88dd35db5be5c83"

  - `replicationPartners.replicationPartnerSystemId` (string,null)
    SystemId of target replication partner
    Example: "7CE816P0SR"

  - `replicationPartners.targetReplicationId` (string,null)
    Id of target replication partner
    Example: "afb4961e47212e5bc88dd35db5be5c83"
