# Get licenses of the storage system identified by {systemId}

Get licenses of the storage system identified by {systemId}

Endpoint: GET /storage-fleet/v1alpha1/devtype4-storage-systems/{systemId}/licenses
Version: 1.2.0
Security: bearer
