# HPE GreenLake for Tags Internal APIs

With the HPE GreenLake for Tags Internal APIs, you can insert or update supported tagged resources with tags in a workspace. <br><br> **NOTE:** This API is available internally for HPE employees only.

Version: v1
License: HPE License

## Servers

HPE GreenLake URL for accessing Tags Internal APIs
```
https://aquila-app-api.common.cloud.hpe.com
```

## Security

### Bearer

Type: http
Scheme: bearer
Bearer Format: JWT

 - [PUT /tags/v1/tag-resources/upsert](https://developer.greenlake.hpe.com/docs/greenlake/services/tags/public/openapi/nbapi-tags/openapi-internal/tags/upserttagresourcesv1.md): Insert or update a list of tagged resources for a workspace.
