# HPE GreenLake for Tags APIs

With the HPE GreenLake for Tags APIs, you can view and filter all tags and all supported tagged resources in a workspace.

Version: v1
License: HPE License

## Servers

HPE GreenLake URL for accessing Tags APIs
```
https://global.api.greenlake.hpe.com
```

## Security

### Bearer

Type: http
Scheme: bearer
Bearer Format: JWT

[HPE GreenLake for Tags APIs](https://developer.greenlake.hpe.com/_bundle/docs/greenlake/services/tags/public/openapi/nbapi-tags-latest/openapi.yaml)

## Tags - v1

### Get tags associated to a workspace

 - [GET /tags/v1/tags](https://developer.greenlake.hpe.com/docs/greenlake/services/tags/public/openapi/nbapi-tags-latest/openapi/tags-v1/gettagsv1.md): Retrieve a list of tags associated to a workspace. Various query parameters can be passed to filter, limit, or sort the returned tags.

### Get all the tagged resources for a workspace

 - [GET /tags/v1/tag-resources](https://developer.greenlake.hpe.com/docs/greenlake/services/tags/public/openapi/nbapi-tags-latest/openapi/tags-v1/gettagresourcesv1.md): Retrieve all the tagged resources for a workspace. Various query parameters can be passed to filter, limit, or sort the returned tagged resources.
