# Refresh the Software Catalog versions available for the system identified by id.

Refresh the software catalogs and software versions on the specified system.
The current and available software catalog versions are recomputed based on the component versions running on the system and software catalogs available for update.

Endpoint: POST /private-cloud-business/v1beta1/systems/{id}/software-version-refresh
Version: 1.1.0
Security: bearer

## Path parameters:

  - `id` (string, required)
    Unique Identifier of the system, usually a UUID.
