# Resume the failed and paused software update on the selected System.

Resume the failed and paused software update on the specified system.
Invoke this API only after fixing the problems reported in software update.

Endpoint: POST /private-cloud-business/v1beta1/systems/{id}/software-update-resume
Version: 1.1.0
Security: bearer

## Path parameters:

  - `id` (string, required)
    Unique Identifier of the System, usually a UUID.
