# Getting started with HPE GreenLake cloud APIs

To help you get started with HPE GreenLake APIs and the HPE GreenLake Developer Portal, this page provides a curated listing of HPE Developer Community events, blogs, and labs to help.

## HPE Developer Community Events

[HPE Developer Community Events](https://developer.hpe.com/events) are opportunities to hear from leading HPE and industry technologists.

We recommend the following events for developers interested in using HPE GreenLake APIs.

### Exploring HPE GreenLake APIs

This presentation shows you how to automate the integration of HPE GreenLake services into your custom applications to streamline processes and enhance control and visibility over your infrastructure.

iframe
### Enabling business automation using HPE GreenLake APIs

This event explores using the HPE GreenLake Developer Portal to access HPE GreenLake APIs, and demonstrates using an OpenAPI tool (such as Postman). The event also provides examples of some of the most popular scripting languages, such as Bash, PowerShell, and Python.

iframe
## HPE Developer Community Blogs

The [HPE Developer Community Blogs](https://developer.hpe.com/blog/?activeTab=1) include blog posts from HPE and other technologists covering an array of topics.
If you want to get using the HPE GreenLake APIs, the following blog posts will help you on your way.

### Getting started with HPE GreenLake Developer Portal

Not sure where to start with the HPE GreenLake Developer Portal? This blog post introduces the portal, how to navigate it, and how to get the most out of your time browsing the portal. [Read the blog](https://developer.hpe.com/blog/getting-started-with-the-hpe-greenlake-developer-portal/).

### Getting started with the foundational APIs

This series guides IT administrators in automating IT operations programmatically using HPE GreenLake service APIs.

1. [Introduction to the APIs](https://developer.hpe.com/blog/get-started-with-the-foundational-apis-for-the-hpe-greenlake-edge-to-cloud-platform-%E2%80%93-part-1-introduction-to-the-apis/)—Part one describes some of the available APIs and takes you through the steps to make your first API calls to HPE GreenLake services using Postman.
2. [Configuring and managing a workspace](https://developer.hpe.com/blog/get-started-with-the-foundational-apis-for-the-hpe-greenlake-edge-to-cloud-platform-%E2%80%93-part-2-configuring-and-managing-a-workspace/)—Part two shows how a Workspace administrator can programmatically configure and manage workspace resources in a single tenant environment.
3. [Tracking activities and monitoring health](https://developer.hpe.com/blog/get-started-with-the-foundational-apis-for-the-hpe-greenlake-edge-to-cloud-platform-%E2%80%93-part-3-tracking-activities-and-monitoring-health/)—Part three follows on logically, and explains how to track and monitor the health of HPE services and products.


### Getting started with scripting

This collection of community blogs explain, through practical examples, how to build scripts to perform common HPE GreenLake tasks.

- [HPE GreenLake cloud scripting fundamentals](https://developer.hpe.com/blog/hpe-greenlake-edge-to-cloud-platform-scripting-fundamentals/)—This blog post explains how to use the HPE GreenLake APIs to build automation scripts and custom integrations using Bash, Python, and PowerShell.
- [Bulk onboarding of users in HPE GreenLake cloud](https://developer.hpe.com/blog/bulk-onboarding-of-users-in-hpe-greenlake-edge-to-cloud-platform/)—Learn how to use PowerShell scripts to onboard users in bulk.


### HPE Sustainability Insight Center

HPE GreenLake provides a new service called HPE Sustainability Insight Center that can assist you in obtaining detailed information about carbon emissions, energy consumption and the cost of the infrastructure that is managed by HPE GreenLake. This blog describes the [HPE Sustainability Insight Center API at a Glance](https://developer.hpe.com/blog/hpe-sustainability-insight-center-api-at-a-glance/)

## Hands-on labs

Take advantage of our free, Jupyter-Notebook-based Workshops-on-Demand available in the [Hack Shack](https://developer.hpe.com/hackshack/workshops/?activeTab=4). Designed to fit your schedule, these workshops are available 24/7–any time, from anywhere. HPE GreenLake API workshops are available today.
Check out the [Get started with the HPE GreenLake foundational API](https://developer.hpe.com/hackshack/workshop/47/) lab.
