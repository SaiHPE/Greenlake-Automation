# Delete a group

Delete a group.

Endpoint: DELETE /compute-ops-mgmt/v1/groups/{group-id}
Version: latest
Security: Bearer

## Path parameters:

  - `group-id` (string, required)

## Query parameters:

  - `force` (boolean)
    If you would like to remove all devices and delete the group, provide the query parameter force=true.
