# Delete a host by {hostId}

Delete a host by {hostId}

Endpoint: DELETE /block-storage/v1alpha1/host-initiators/{hostId}
Version: 1.0.0
Security: bearer

## Path parameters:

  - `hostId` (string, required)
    Id of the Host.
    Example: "2b09e744496246859fde6c132b2091d3"

## Query parameters:

  - `force` (boolean)
    Forceful delete option
    Example: true

  - `delete-associated-empty-hg` (boolean)
    Delete the resulting empty host group associated to this host option
    Example: true
