# Get volume-set identified by id

Get volume-set identified by id

Endpoint: GET /block-storage/v1alpha1/storage-systems/{systemId}/volume-sets/{id}
Version: 1.0.0
Security: bearer

## Path parameters:

  - `systemId` (string, required)
    systemId of the storage system
    Example: "7CE751P312"

  - `id` (string, required)
    UID of Volume Set
    Example: "fd3244ef7f1ab8bd16500c7a41bdf8f8"

## Query parameters:

  - `select` (string)
    Query to select only the required parameters, separated by . if nested
    Example: "id"
