# Create a trust certificate for storage cluster

This API endpoint is to create a trust certificate for storage cluster

Endpoint: POST /storage-fleet/v1alpha1/devtype7-storage-systems/{systemId}/trust-certificates
Version: 1.2.0
Security: bearer

## Path parameters:

  - `systemId` (string, required)
    ID of the storage system
    Example: "2a0df0fe6f7dc7bb16000000000000000000004817"

## Request fields (application/json):

  - `trustStoreRef` (string, required)
    The name of the applicable Trust Store.
    Example: "Audit-default-trust-store"

  - `certificate` (object, required)
    User provided info for trust certificate creation.

  - `certificate.certificate` (string, required)
    Certificate

  - `certificate.name` (string, required)
    The certificate name
    Example: "certificate-1"
