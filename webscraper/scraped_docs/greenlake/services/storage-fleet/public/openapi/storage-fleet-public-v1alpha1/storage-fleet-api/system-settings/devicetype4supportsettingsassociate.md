# Add support settings for an HPE Alletra Storage MP B10000 storage system

Add support settings for an HPE Alletra Storage MP B10000 storage system

Endpoint: POST /storage-fleet/v1alpha1/devtype4-storage-systems/{systemId}/support-settings
Version: 1.2.0
Security: bearer

## Path parameters:

  - `systemId` (string, required)
    systemId of the storage system
    Example: "7CE751P312"

## Request fields (application/json):

  - `connectToHpe` (string, required)
    Enable remote support by allowing sending of files from device to HPE. Allowed values: enabled or disabled. It is mandatory.

  - `remoteAccess` (string, required)
    Allow HPE Support to access the device remotely. Allowed values: ENABLE_ROOT or DISABLE or ENABLE_NONROOT. It is mandatory.
    Enum: "DISABLE", "ENABLE_NONROOT", "ENABLE_ROOT"

  - `deviceId` (string)
    Id of the array. User can get Id info from GET response and this can not be changed.

  - `enterpriseServerUrl` (string,null)
    Callhome collection server URL

  - `miniInsploreEnabled` (string,null)
    Enables/Disable scheduled Mini-Insplore collection. Allowed values: enabled or disabled.

  - `remoteRequestAcknowledge` (object,null)
    Accept/Deny/Close the Remote Access request.

  - `remoteRequestAcknowledge.id` (string)
    Id of the person requesting Remote Access

  - `remoteRequestAcknowledge.supportEmail` (string,null)
    Mail id of the person requesting Remote Access

  - `remoteRequestAcknowledge.userAction` (string,null)
    Accept/Deny Remote Access Request
    Enum: "ACCEPT", "DENY", "CLOSE"
