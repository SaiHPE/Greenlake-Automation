# Import certificate identified by {id} on HPE Alletra Storage MP B10000 storage system identified by {systemId}

Import certificate identified by {id} on HPE Alletra Storage MP B10000 storage system identified by {systemId}

Endpoint: PUT /storage-fleet/v1alpha1/devtype4-storage-systems/{systemId}/certificates/{id}
Version: 1.2.0
Security: bearer

## Path parameters:

  - `systemId` (string, required)
    systemId of the storage system
    Example: "7CE751P312"

  - `id` (string, required)
    ID of the certificate
    Example: "99691e493067b2b2acf1774fc0ccc011"

## Request fields (application/json):

  - `authorityChain` (string, required)
    The authority chain for the new certificate.
    Example: "-----BEGIN CERTIFICATE REQUEST-----abc----END CERTIFICATE REQUEST----- \\n-----BEGIN CERTIFICATE REQUEST-----pqr----END CERTIFICATE REQUEST-----"

  - `certificate` (string, required)
    The certificate that results from the Certificate Signing Request (CSR).
    Example: "-----BEGIN CERTIFICATE REQUEST-----abc----END CERTIFICATE REQUEST-----"

  - `vcGuid` (string,null)
    vcGuid of the vCenter. It is supported from OS version 10.4.0 and VASA version 5.0.0.0
    Example: "09r907r9hinfniauhfiqh98qh"
