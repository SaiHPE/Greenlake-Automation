# Add trusted certificates for HPE Alletra Storage MP B10000 storage system identified by {systemId}

Add trusted certificates for HPE Alletra Storage MP B10000 storage system identified by {systemId}

Endpoint: POST /storage-fleet/v1alpha1/devtype4-storage-systems/{systemId}/trust-certificates
Version: 1.2.0
Security: bearer

## Path parameters:

  - `systemId` (string, required)
    systemId of the storage system
    Example: "7CE751P312"

## Request fields (application/json):

  - `action` (string, required)
    Action to perform with trusted certificate
    Enum: "VERIFY_CERT_CHAIN", "INSTALL_CERT_CHAIN", "VERIFY_AND_INSTALL_CERT_CHAIN"

  - `parameters` (object, required)
    Parameters for adding a trusted certificate

  - `parameters.type` (string, required)
    Type of the trusted certificate
    Example: "other"

  - `parameters.certificates` (string, required)
    Trusted certificate text
    Example: "----BEGIN CERTIFICATE -----abc----END CERTIFICATE -----"
