# Edit settings of a HPE Alletra Storage MP X10000 system Storage Node  identified by storageNodeID

Edit settings of a HPE Alletra Storage MP X10000 system Storage Node identified by storageNodeID

Endpoint: PUT /storage-fleet/v1alpha1/devtype7-storage-systems/{systemId}/storage-nodes/{storageNodeId}
Version: 1.2.0
Security: bearer

## Path parameters:

  - `systemId` (string, required)
    ID of the storage system
    Example: "2a0df0fe6f7dc7bb16000000000000000000004817"

  - `storageNodeId` (string, required)
    Identifier of StorageNode. A 42 digit hexadecimal number.
    Example: "2a0df0fe6f7dc7bb16000000000000000000004007"

  - `frontendNics` (array,null)

  - `frontendNics.ports` (array,null)

  - `frontendNics.slotNumber` (integer,null)
