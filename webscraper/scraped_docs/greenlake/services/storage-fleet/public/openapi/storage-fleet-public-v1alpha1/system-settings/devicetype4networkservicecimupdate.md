# Edit CIM network service configuration

Edit CIM network service configuration

Endpoint: PUT /storage-fleet/v1alpha1/devtype4-storage-systems/{systemId}/cim
Version: 1.2.0
Security: bearer

## Path parameters:

  - `systemId` (string, required)
    systemId of the storage system
    Example: "7CE751P312"

## Request fields (application/json):

  - `cim` (object,null)

  - `cim.cimPolicy` (boolean,null)
    Specifies the CIM Policy of CIM server.
    Example: true

  - `cim.enableCimService` (boolean,null)
    Enable or disbale Cim service.
    Example: true

  - `cim.httpState` (boolean,null)
    Specifies whether HTTPState is enabled or disabled for CIM Server. Unsupported from version 10.4.0 and later
    Example: true

  - `cim.httpsState` (boolean,null)
    Specifies whether HTTPS state is enabled or disabled for cim server. Unsupported from version 10.4.0 and later
    Example: true

  - `cim.id` (string)
    Unique identifier of the CIM Server.

  - `cim.slpState` (boolean,null)
    SLPport specification.
    Example: true
