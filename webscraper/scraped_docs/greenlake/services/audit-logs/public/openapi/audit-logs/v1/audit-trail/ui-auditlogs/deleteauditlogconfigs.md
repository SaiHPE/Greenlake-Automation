# Delete external application audit logs specific configurations.

Endpoint: DELETE /auditligs/ui/v1/configs/{audit-id}
Version: v1
Security: BearerAuth, CookieAuth

  - `audit-id` (string, required)
    Audit id

  - `message` (string)
    Audit Id.
