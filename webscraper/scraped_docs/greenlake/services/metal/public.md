# HPE GreenLake for Bare Metal

The HPE GreenLake for Bare Metal service facilitates provisioning and managing computing and storage on Bare Metal systems in HPE GreenLake Central. Using this service, customers can configure containers and virtual machines to get the best available option that suits their business. This customization offers customers systems tailored perfectly for their use at economical costs.
