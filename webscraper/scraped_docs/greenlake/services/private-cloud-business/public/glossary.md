# HPE GreenLake for Private Cloud Business Edition

## Acronyms

### dHCI

A disaggregated HCI product sold by Hewlett Packard Enterprise (HPE). It is composed of a Storage Group, a system of ProLiant servers, and a set of top-of-rack switches.

### HCI

Hyperconverged infrastructure (HCI) is a software-defined, unified system that combines all the elements of a traditional data center: storage, compute, networking and management. HPE has introduced the concept of disaggregated HCI to denote that the storage and compute can be scaled independently. This simply means the storage data services are not running virtualized but on their own hardware.

## Terms

### Provisioning policy

Provides a wrapper on Storage Policy Based Management(SPBM). It encapsulates volume attributes and protection policy.
It provides VM level protection and disk level recovery through Backup and Recovery integrations.

### System software catalog or simply 'catalog'

Refers to a 'Recipe' that is a collection of software components validated to work together. Private Cloud Business Edition software update service uses these catalogs to identify the current and available catalogs for software update on a per hypervisor cluster basis. Each catalog is assigned a version based on the versions of constituent software components like storage software, hypervisor cluster and server firmware.
