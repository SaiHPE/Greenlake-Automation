# Initiate software update on the given System.

Initiate a software update on the specified system and the hypervisor clusters in it using the specified software catalog version.
Before invoking this API, ensure that software update prechecks on the same parameters successfully completed and is valid.

Endpoint: POST /private-cloud-business/v1beta1/systems/{id}/software-update
Version: 1.1.0
Security: bearer

## Path parameters:

  - `id` (string, required)
    Unique Identifier of the System, usually a UUID.

## Request fields (application/json):

  - `catalogVersion` (string, required)
    Version of the target software catalog for the software precheck/update request.

  - `hypervisorClusterIds` (array)
    List of Unique Identifiers (usually UUIDs) of Hypervisor Clusters in the system for which software update operation is to be initiated. If omitted, software update operation will be initiated on all Hypervisor Clusters available in the system.
